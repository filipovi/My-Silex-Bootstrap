Silex bootstrap using Composer packages
=======================================

Packages included :

- Twig
- Monolog
- Doctrine/Common
- Doctrine/DBAL
- Symfony/Twig-Bridge
- Symfony/Form
- Symfony/Validator
- Symfony/Translation
- Symfony/YAML
- Symfony/Config

Install
------

To install it :

`git clone`

and

`php composer.phar install --verbose`

Tests
-----

`phpunit`

and

`behat`
